Git Branching

5 Types of Branching

1.) Master Branch: Main branch where the final and working version of the code will go for a live site

2.) Release Branch: Secondary Branch that verifies new code before being released on Master

3.) Develop Branch: Branch where the majority of issues are corrected and merged from the class branch

4.) Class Branch: Code for a paticular class

5.) Hot Branch: A branch for patches that is typically only done when an enterprise is losing income
